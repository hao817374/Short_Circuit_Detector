# Short Circuit Detector v1.0.8 版本说明

## 缺陷修复

- **"清除校准"按钮零点校准数据残留**:
    - 修复点击"清除校准"后零点校准结果卡片（Q0/Q1/BIAS 数值）及状态文字未被清除的问题。
    - `resetCalibration()` 中新增 `setZeroCalibResult(null)` 和 `setZeroCalibStatus('IDLE')`，确保零校准 UI 与方向校准 UI 同步清除。

## 交互优化

- **方向校准按钮顺序调整**:
    - CALIBRATION 页面方向校准卡片中，"右上角激励源"按钮移至上方，"右下角激励源"按钮移至下方，与开发板物理布局一致。

- **导航栏设置按钮样式统一**:
    - 系统配置按钮从单一图标改为与"方位检测""空间校准"一致的中文+英文双行结构。
    - 导航栏整体字体提升：中文 10px→11px，英文 8px→10px。

- **空间校准页面排版优化**:
    - 卡片标题字号 `text-sm`→`text-base`，说明文字 `text-xs`→`text-sm`，强化信息层级。
    - 方向校准按钮标签 `text-[10px]`→`text-xs`，底部操作按钮同步提升，改善可读性。
    - 零点校准按钮宽度增加 `max-w-[240px]` 约束，避免与结果卡片等分空间。
    - 零校准/方向校准结果数值 `text-xs`→`text-sm`，数据更清晰。

- **空间校准页面布局重构**:
    - 方向校准按钮文字精简："请将表笔点在右下角激励源"→"右下角激励源"（11→6 字）。
    - 左右面板比例 `flex-[2]`:`flex-1`（66:33）→ `flex-1`:`flex-1`（50:50），波形预览区域更大。
    - 面板/卡片 padding 响应式优化：`p-10`→`p-6 lg:p-8`，`p-8`→`p-6`。
    - 罗盘 `min-h-[70vh]`→`min-h-0`，修复默认 1200×800 窗口下罗盘不能完整显示的问题。

- **方向校准按钮宽度与数值标签优化**:
    - 方向校准按钮增加 `max-w-[360px]` 约束，与零点校准按钮的 `max-w-[240px]` 逻辑一致。
    - 校准结果数值标签格式统一："Q0"→"Q0:"，"Q1"→"Q1:"，"BIAS"→"BIAS:"，数字紧贴标签，去除固定宽度的 `w-14`/`w-10`。
    - 窗口默认高度 800→1000，提供更充裕的纵向空间。

- **系统配置页面基础显示设置优化**:
    - 全局字号 `text-[10px]`→`text-xs`，覆盖顶部按钮（日间/夜间/中文/EN/重置/开发者模式）及卡片内标签。
    - 滑动条最小值/最大值/步进精度从数字输入改为滑动条控件（范围 5-30/300-500/5-20）。
    - 标签 "初始短路阈值"→"短路阈值"。
    - 布局改为两行两列网格：滑动条最小值↗最大值、短路阈值↘步进精度，警告提示居中置于网格下方。
