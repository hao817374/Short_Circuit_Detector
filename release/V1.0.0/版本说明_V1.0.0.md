# 短路测试仪 (Short Circuit Detector)
## 版本：V1.0.0

### 🎉 发布说明
这是本系统的首个正式 Windows 桌面可执行版本。基于 “软件定义仪器” 的理念，成功实现了对微弱短路磁场信号的高精度捕捉和解析。

### ✨ 核心功能
1. **基于仿射变换的正交重构**：采用 2x2 变换矩阵，完美消除了由主板铺铜不均匀导致的 XY 轴向增益差和几何串扰，指针旋转平滑且线性。
2. **硬件级时分复用解调支持**：通过底层串口直接读取 X 轴、Y 轴相敏检波分离后的正交幅度分量。
3. **极简交互式示波器**：支持用户拖拽采样窗口 (WIN1/WIN2)，自适应不同相位差。

### 🚀 性能优化记录 (Zero-Allocation Refactor)
在此版本发布前，我们对底层的数字信号处理（DSP）数据流进行了微创级性能重构，极大降低了 CPU 占用：
* **消除 V8 引擎热路径垃圾回收 (GC)**：预先分配了 `Float64Array(32)` 作为静态共享内存。去噪算法 `getWindowAvg` 不再每次分配和丢弃零碎数组，实现了核心逻辑的**零内存分配 (Zero-Allocation)**。
* **极速串口数据流解析**：废除了原有的重量级正则表达式解析器（Regex Capture Groups），采用底层的 `split(/\s+/)` 高效拆分字符串，彻底解决了 60Hz 高频通讯下的偶发卡顿问题。

### 📥 安装指南
双击本目录下的 `Short Circuit Detector Setup 1.0.0.exe` 即可自动安装并创建桌面快捷方式。
