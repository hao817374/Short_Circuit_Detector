# 短路测试仪 V1.0.3 发行说明

此版本完成了对 UI 界面和功能布局的深度重构，旨在提升工业作业效率并隔离专业调试功能。

## ✨ UI/UX 重大更新

### 1. 统一导航架构 (Unified Navigation)
- **胶囊式导航栏**：顶部导航已整合为统一组，减少视觉碎片化。
- **专业中文标识**：采用“大字中文 + 小字英文”布局，更符合国内工程师使用习惯。
- **校准一级页面化**：将“空间校准”从二级弹窗提升为一级作业页面。由于每个板卡测试前都需要校准，此改动极大提升了作业流畅度。

### 2. 开发者模式 (Developer Mode)
- **角色功能隔离**：新增“开发者模式”开关（位于系统配置中）。
- **按需显示**：默认状态下隐藏“调试分析”页面和高级算法参数，避免普通用户误触。
- **高级调试**：开启后可访问实时波形视图、ADC 原始数据偏移量调节等核心参数。

## 🚀 优化与细节

### 3. 校准体验增强
- **实时波形反馈**：校准页面现在拥有更大的实时波形显示区域。
- **引导式操作**：优化了采样按钮的视觉状态和进度反馈。

### 4. 界面清理
- 移除多余的顶部常驻按钮，使界面中心更加聚焦于当前作业。
- 优化了“系统配置”的布局，将设置项划分为“基础显示”与“高级算法”两个分区。
