## 版本：V4.1.0 | 状态：正式发布

## 1. 绪论：项目背景与行业痛点 (深度解析)

### 1.1 电子工业微型化演进与维修挑战的指数级增长
在过去的三十年中，摩尔定律不仅驱动了半导体性能的飞跃，也彻底改变了印刷电路板（PCB）的形态。从早期的单面板、双面板，演进到如今动辄 10 层、16 层甚至更高层数的 HDI（高密度互连）板材，电子线路的集成度已达到了物理极限的边缘。这种微型化趋势带来了前所未有的维修挑战。

在传统的通孔插装时代，维修工程师可以凭借肉眼或简单的万用表轻松追踪信号路径。然而，随着 01005 封装元件的普及、微盲孔技术以及嵌入式无源元件的大规模应用，PCB 内部的电流路径变得如同迷宫般复杂。当这些高度集成的电路板发生电源轨（Power Rails）短路故障时，故障点往往隐藏在多层结构的内部，或掩盖在巨大的 BGA 封装芯片下方。传统的探测手段在这种“暗盒”结构面前显得苍白无力。

### 1.2 短路故障的物理本质与破坏性扩散
短路不仅仅是一个电气连接错误，它在微观层面是一个复杂的能量耗散过程。在电源管理芯片（PMIC）驱动的现代主板中，核心供电（VCORE）通常具有超大电流、极低电压（低至 0.8V）的特点。当该支路发生短路时，故障点的阻抗通常在毫欧（mΩ）级别。

如果没有精密的定位工具，维修人员通常不得不采取“逐一拆解法”，即通过拆除支路上的所有滤波电容来排查短路点。这种方法不仅效率极低，更会对 PCB 焊盘造成不可逆的热损伤，甚至导致电路板报废。更危险的“烧机法”——即向短路支路注入大电流以寻找发热点——在面对 7nm 或更先进制程的 CPU/GPU 时，极易因过压或电流浪涌直接击穿处理器内部的脆弱晶体管，导致原本几百元的维修变成几千元的报废损失。

### 1.3 传统诊断方法的局限性深度剖析

#### 1.3.1 万用表阻抗测量的失效
万用表是电子工程师的基石，但在短路定位上，它存在致命的“分辨率瓶颈”。标准万用表的最小分辨率通常为 0.01Ω 或 0.1Ω。在一条布满了几十颗贴片电容的电源线上，万用表在任何一个电容两端测得的阻抗几乎都是一致的，因为它无法分辨出由于几厘米铜箔走线产生的微小寄生电阻差异。

#### 1.3.2 红外热成像的“热扩散”干扰
红外热成像（Infrared Thermography）是近年来流行的手段，但它受限于物理定律。短路产生的热量会迅速通过大面积的铺铜地层（Ground Plane）扩散。在多层板中，顶层看到的热点往往只是底层热源通过过孔传导上来的“虚像”。此外，如果短路点位于 BGA 芯片下方，热成像根本无法穿透封装外壳观察到内部的异常。

#### 1.3.3 冰雾/松香雾法的经验主义局限
传统的松香雾法依赖于目测由于发热导致的物性变化。这种方法不仅污染电路板（残留助焊剂），且灵敏度极低，仅适用于电流极大的硬短路，对于电流较小的微短路或漏电故障束手无策。

### 1.4 软件定义仪器（SDI）的工程逻辑
面对上述痛点，本项目提出了一套基于“软件定义仪器”逻辑的解决方案。我们意识到，硬件传感器的精度固然重要，但对复杂信号的实时解析、噪声过滤以及矢量合成，才是决定定位精度的关键。

Short Circuit Detector V4.1.0 系统的核心哲学是：**将不可见的磁场转化为可视化的矢量导航**。
通过采集短路电流在空间中产生的微弱磁感应强度（B），利用高性能 MCU 进行模数转换，并由本软件通过复杂的数学模型进行逆向推导，从而在屏幕上为维修工程师提供一个实时的、“GPS 级别”的定位罗盘。这种非侵入式（Non-invasive）、非破坏性的检测方法，代表了精密电子维修的未来方向。

### 1.5 全球维修经济与环保意义
从宏观角度看，精密短路定位工具的普及具有重大的环保和经济意义。每年全球产生数千万吨电子垃圾（E-waste），其中相当一部分是因为“无法定位短路点”而被遗弃的高价值主板。通过本系统提高修复率，不仅为企业和个人节省了数以亿计的更换成本，更通过延长电子产品寿命，显著降低了电子垃圾的产生，符合可持续发展的工业设计理念。

## 2. 系统物理建模与数学基础

### 2.1 毕奥-萨伐尔定律的应用
短路点会形成一个封闭的电流回路。根据毕奥-萨伐尔定律，载流导线周围会产生磁场。探头的两个正交线圈（Q0 和 Q1）采集该磁场的垂直分量：
- **Q0 (通道 A)**: 采集 X 轴向磁感应强度。
- **Q1 (通道 B)**: 采集 Y 轴向磁感应强度。

### 2.2 矢量合成模型
软件将 Q0 和 Q1 视为笛卡尔坐标系中的两个矢量分量：
- **方位角 (Heading)**: $\theta = \text{atan2}(Q1, Q0)$
- **信号模值 (Magnitude)**: $M = \sqrt{Q0^2 + Q1^2}$
模值的大小直接反映了探针距离短路路径的远近。

## 3. 软件架构设计 (Software Architecture)

### 3.1 零延迟异步通讯架构
系统基于 **Web Serial API** 构建，采用生产者-消费者模型。
- **生产者 (Serial Loop)**: 运行在后台的异步循环，通过 `reader.read()` 不断拉取原始字节流。
- **中间件 (Parser)**: 将字节流解码为 UTF-8 文本，并利用高性能正则表达式 `/^(\d+)\s+(-?\d+)\s+(\d+)\s+(\d+)$/` 进行原子化拆解。
- **消费者 (DSP Processor)**: 每当累积 138 个点时，触发帧提交逻辑。

### 3.2 React 渲染性能调优
在处理 60Hz 采样率的图形渲染时，普通的 React 状态管理会导致严重的 UI 卡顿。
- **Ref-Mirroring 机制**: 所有的计算参数（如偏置、补偿系数）同时存在于 `useState` 和 `useRef` 中。计算引擎只读取 `useRef` 以避免重新渲染开销，而 UI 组件订阅 `useState` 进行间歇性更新。
- **SVG Layering**: 示波器背景线、静态遮罩、动态波形分层渲染，大幅减少了 SVG DOM 的重绘范围。

## 4. 核心信号处理算法 (Deep Dive in DSP)

### 4.1 窗口化修剪平均值算法 (Windowed Trimmed Mean)
这是本系统最关键的降噪算法。
1. **采样窗口选取**: 系统在每帧 138 个点中截取指定的 32 点作为运算基准。
2. **数据清洗**:
   - 排序：将 32 个点按电压值升序排列。
   - 修剪：剔除前 6 个（低谷噪声）和后 6 个（尖峰干扰）。
3. **均值计算**: 计算剩余 20 个点的算术平均值。
**为什么这样做？** 
探针在滑动过程中会产生由于摩擦电效应引起的瞬态毛刺。传统的滤波（如简单移动平均）会产生明显的相位延迟，而修剪平均值能在保持极低延迟的同时，彻底消除非高斯分布的随机噪声。

### 4.2 2点轴平衡补偿算法 (The 2-PT Calibration)
由于硬件探头在生产中存在阻抗不匹配，Q0 和 Q1 的灵敏度往往不一致。
1. **基准采样**: 在 NW(45°) 和 SW(135°) 进行采样。
2. **灵敏度比例计算**: 
   $$K = \frac{\max(V_{NW}, V_{SW})}{\min(V_{NW}, V_{SW})}$$
3. **动态增益映射**: 
   程序会自动识别响应较弱的一路。在后续计算中：
   $$Value_{weak} = RawValue_{weak} \times K$$
这确保了“逻辑罗盘”的指向与“物理方向”完全 1:1 映射。

## 5. GUI 与人机交互设计 (UX Engineering)

### 5.1 HUD 指向系统逻辑
罗盘组件 `Compass.tsx` 不仅是一个图形，它是一个复杂的状态机：
- **插值平滑**: 通过 `visualHeading` 计算最短旋转路径（Shortest Rotation Path），避免指针从 359° 旋转到 1° 时发生逆时针大转。
- **环境光渲染**: 针对不同的状态（Navigating / Nearby / Disconnected），SVG 背景会切换不同的径向渐变（Radial Gradient）和滤镜（Blur Filter），利用视觉色彩心理学提示维修人员当前的操作状态。

### 5.2 交互式示波器
- **拖拽式窗口设置**: 在调试界面，用户可以直接鼠标拖拽 `WIN1` 和 `WIN2`。这本质上是在动态调整数组的切片索引（Slice Index），使得软件能自适应不同主板上激励信号的相位差。
- **自适应垂直刻度 (Auto-Scaling)**: 系统实时监控 $V_{peak-to-peak}$，动态计算 SVG `viewBox` 的垂直范围。

## 6. 系统安全性与稳定性保障

### 6.1 串口死锁防护
Web Serial 通讯极易因为设备拔出而导致浏览器渲染进程死锁。本软件实现了：
- **三向握手断开逻辑**: `reader.cancel()` -> `reader.releaseLock()` -> `port.close()`。
- **Ref-Based 锁机制**: 确保在端口关闭过程中，异步的 `readLoop` 不会再次尝试访问空句柄。

### 6.2 异常数据过滤
当硬件探测到极大电流干扰（例如探针短路或接触高压）时，电压值会跳变至 > 20000。软件会自动进入 `PROBE DISCONNECTED` 保护模式，清空罗盘并锁定指针，防止误导用户。

## 7. 总结与展望

### 7.1 现有成果
Short Circuit Detector V4.1.0 成功将数字信号处理与 Web 交互技术深度融合。其 138 点/帧的解析精度和 2 点平衡补偿技术，已达到工业级精密仪器的标准。

### 7.2 未来路线
1. **FFT 频谱分析**: 未来将引入快速傅里叶变换，分析信号的谐波成分，从而判断 PCB 上的干扰源是开关电源还是晶振噪声。
2. **AI 自动相位搜索**: 引入机器学习模型，自动识别波形的相位特征，实现零配置开箱即用。
3. **移动端适配**: 利用 WebUSB 协议将系统迁移至移动端，实现更便携的操作。
